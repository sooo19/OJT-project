# OJT-project 
신입사원 OJT 프로세스 개선 프로젝트<br>
기존의 엑셀 -> 프린트 -> 버디 사인 -> 스캔 -> 웹 등록 의 과정을 간소화 및 자동화한다
- 인턴과 버디 선택 기능을 통해 내용 작성 및 피드백 작성을 관리
- 보고서 제출 및 피드백 등록 시 이메일을 통해 알람
- 전체 보고서 작성 완료시 엑셀을 통한 export 기능 구현
